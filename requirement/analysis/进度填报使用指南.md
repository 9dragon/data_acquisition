# 进度填报功能使用指南

## 📋 功能概述

本系统支持**三种进度填报方式**，覆盖不同场景的填报需求：

1. **按设备填报** - 详细填报单个设备的各阶段进度
2. **快速填报** - 快速更新单个设备的进度（弹窗方式）
3. **批量填报** - 批量更新多台设备在同一阶段的进度

---

## 方式一：按设备填报（详细填报）

### 适用场景
- 需要详细填写单个设备的4个实施阶段进度
- 填写详细的阶段说明和总体说明

### 填报入口
1. **项目计划页面** → 设备进度卡片 → "填报"按钮
2. **设备列表页面** → 操作列 → "填报进度"按钮
3. **直接访问**：`/plan/report/{设备ID}`

## 方式二：快速填报（弹窗）

### 适用场景
- 快速更新单个设备的进度
- 只需要更新当前正在进行的某个阶段
- 填写简要说明

### 填报入口
**项目计划页面** → 顶部"快速填报"按钮

### 操作步骤
1. 点击"快速填报"按钮
2. 弹窗显示第一台设备信息
3. 选择要填报的阶段（显示当前进度）
4. 输入新进度百分比
5. 填写阶段说明和总体说明
6. 点击"提交"直接生效

## 方式三：批量填报（按阶段）

### 适用场景
- 阶段性统一填报，如"施工阶段全部完成"
- 多台设备在同一阶段有相同的进度
- 批量更新提高效率

### 填报入口
1. **项目计划页面** → 顶部"批量填报"按钮 → 选择阶段
2. **项目计划页面** → 底部"批量填报快捷入口" → 直接选择阶段

### 操作步骤
1. 选择批量填报的阶段（如"施工阶段"）
2. 勾选需要填报的设备（支持全选）
3. 设置统一的进度百分比
4. 填写批量填报说明
5. 填写总体说明
6. 点击"提交批量进度"直接生效

## 🎯 使用建议

### 场景一：日常填报
推荐使用**快速填报**
- 每天或定期更新单个设备的进度
- 填报当前正在进行的阶段
- 简洁高效

### 场景二：阶段性节点
推荐使用**批量填报**
- 某个阶段全部完成时
- 多台设备进度相同时
- 批量更新节省时间

### 场景三：详细记录
推荐使用**按设备填报**
- 项目重要节点需要详细记录
- 需要填写所有4个实施阶段的进度
- 审计和追溯需要

## 📝 填报规则

### 必填项
- ✅ 阶段进度百分比（0-100%）
- ✅ 阶段完成情况说明
- ✅ 总体说明

### 生效机制
- ⚡ **直接生效**：填报后立即更新进度，无需审核
- 📊 **实时更新**：进度数据实时同步到项目计划
- 📝 **历史记录**：所有填报记录都会保存，可追溯

### 实施阶段
根据需求文档，项目计划仅针对实施阶段：

1. **准备阶段** (preparation)
   - 人员配置、设备采购、工具准备、技术方案确认

2. **施工阶段** (construction)
   - 设备安装、网络布线、硬件调试、现场测试

3. **配置阶段** (configuration)
   - 点位配置、协议配置、状态逻辑配置、采集测试

4. **核对阶段** (verification)
   - 数据核对、准确性验证、完整性检查、问题修复

## ❓ 常见问题

### Q1: 三种填报方式可以混用吗？
**A:** 可以。系统会自动合并所有填报数据，以最新的填报为准。

### Q2: 填报后可以修改吗？
**A:** 可以。重新填报会覆盖之前的进度数据。

### Q3: 批量填报时可以选择部分设备吗？
**A:** 可以。批量填报支持选择部分设备进行填报。

### Q4: 必须填写所有4个阶段吗？
**A:** 不必。按设备填报需要填写所有阶段，但快速填报只需填写当前阶段。

### Q5: 填报后多久生效？
**A:** 填报后立即生效，无需等待审核。
